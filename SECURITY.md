# Security Policy

## Supported Versions

Software Compass does not yet follow a formal versioning/release schedule.
Security fixes are applied to the latest state of the `main` branch.

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, use GitHub's private
[vulnerability reporting feature](../../security/advisories/new) for this
repository, if enabled, or contact the maintainers directly through a private
channel listed on the repository's GitHub page.

Please include:

* A description of the vulnerability and its potential impact.
* Steps to reproduce, or a proof of concept if possible.
* The affected version/commit.

We aim to acknowledge reports within a few business days and will work with
you on a coordinated disclosure timeline.
